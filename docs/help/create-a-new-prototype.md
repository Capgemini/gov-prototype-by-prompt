# Create a new prototype

Visit the [create a prototype page](/create) to create a new prototype.

Describe what the form is for, what fields it should include, and any eligibility criteria or instructions for the user.

Once created, the high-level structure of the prototype can be previewed in the "Structure" tab.

Visit the [supported prototype options page](/help/supported-prototype-options) for details of the available form and question fields.

## Example prompts

To create a simple driving licence application form:
> Create a driving licence application form. Ask for name, email, date of birth, preferred contact methods, automatic or manual, and to explain any health problems. Must be a UK resident and over 17 years old.

To create a more complex driving licence application form with a branching question and detailed instructions:
> Create a driving licence application form. Ask for name, email, date of birth, automatic or manual, whether they have any declarable health problems, and to explain what they are if they do. Include a detailed explanation for the fifth question that states the user must declare if they have a ‘notifiable’ medical condition or disability that could affect their ability to drive safely. Include a link to <https://www.gov.uk/driving-medical-conditions>

To create a form to apply for a replacement library card:
> Create a form to apply for a replacement library card from Devon County Council. Ask for their name, address, email address, whether they know their previous card number, and if so what the number is.

To create a complex form to register as a PAYE employer:
> Create a form to register a PAYE employer. Ask for employer name, number of employees, if they have a company registration number and if so what that is, and if they provide taxable employee benefits and if so to describe what they are.
